# 마이링크 사용자 시나리오

## 1. 방문자: 마이링크 페이지 접속 및 탐색 과정
- 사용자는 **특정인의 다양한 작업물 및 소셜 채널을 한눈에 확인**하기 위해, **해당 소유자의 고유 마이링크 접속 URL(`domain.com/displayName`)을 눌러 페이지에 접속**한다.
- 사용자는 **이 페이지의 소유자가 누구인지 빠르고 정확하게 파악**하기 위해, **화면 상단에 명시된 표시 이름(displayName), 본명(username), 소개글(Bio) 텍스트를 읽고 확인**한다.
- 사용자는 **자신의 목적에 맞는 타겟 채널로 곧바로 이동**하기 위해, **직관적인 파비콘 아이콘이 부착된 개별 링크 버튼 중 하나를 클릭**한다.

## 2. 소유자: 전체 서비스 활용 과정

### [가입 및 프로필 온보딩]
- 사용자는 **별도의 이메일 가입 절차 없이 빠르고 안전하게 서비스 계정을 생성**하기 위해, **사이트 첫 화면에서 'Google로 로그인' 버튼을 클릭**한다.
- 사용자는 **자신만의 고유한 URL 도메인 주소(슬러그)를 손쉽게 획득**하기 위해, **자동으로 지메일(Gmail) 아이디의 '@' 앞부분이 파싱되어 자신의 표시 이름(displayName)으로 설정되는 현상을 확인**한다.
- 사용자는 **다른 방문자들에게 노출될 프로필 정보를 완성**하기 위해, **본명(username)과 짧은 소개글(Bio)을 채워 넣고 메인 환경인 관리 대시보드로 진입**한다.

### [프로필 정보 인라인 수정]
- 사용자는 **시간이 지남에 따라 변경된 자신의 닉네임(displayName)이나 소개글 등을 최신화**하기 위해, **대시보드 상단에 렌더링된 해당 텍스트를 통째로 클릭하여 인라인 편집 모드로 진입**한다.
- 사용자는 **별도의 회원 정보 관리 페이지로 이동하는 귀찮음 없이 현재 화면에서 즉각적으로 조작을 완료**하기 위해, **텍스트 수정을 마친 후 엔터(Enter) 키를 누르거나 빈 곳을 클릭하여 Firebase DB에 데이터를 실시간 저장**한다.

### [링크 추가]
- 사용자는 **새로운 외부 채널이나 포트폴리오를 대중에게 추가로 공유**하기 위해, **대시보드 메인 화면 하단에 위치한 '+ 링크 추가' 버튼을 찾아 클릭**한다.
- 사용자는 **방문자에게 명확한 이동 목적지(URL 주소)를 안내**하기 위해, **생성된 폼의 빈 슬롯에 대상 '웹 주소(URL)'와 버튼에 쓰일 '표시 제목(Title)'을 타이핑하여 기입**한다.
- 사용자는 **등록한 링크 버튼의 시각적 완성도와 직관성을 향상**시키기 위해, **URL 입력을 마친 직후 구글 파비콘 API 로직이 작동하여 대상 사이트의 로고가 자동으로 버튼 아이콘에 채워지는 것을 확인**한다.

### [링크 수정]
- 사용자는 **이미 등록해 놓은 링크의 잘못된 도메인 주소나 낡은 제목을 최신화**하기 위해, **해당 링크의 텍스트 영역을 마우스로 살짝 클릭하여 딜레이 없는 인라인 폼으로 활성화**한다.
- 사용자는 **다른 설정 화면으로 전환할 필요 없이 즉석에서 조작을 완료**하기 위해, **원하는 텍스트로 데이터를 덮어쓴 뒤 엔터(Enter) 혹은 여백 클릭을 통해 즉시 데이터를 영구 저장**한다.

### [링크 삭제 및 실행 취소 (안전장치)]
- 사용자는 **더 이상 유효하지 않게 된 특정 링크를 방문자들의 시야에서 즉각 차단**하기 위해, **해당 링크 아이템의 우측 끝단에 배치된 '휴지통(삭제)' 아이콘을 단절적으로 클릭하여 비활성**한다.
- 사용자는 **자신의 터치 실수나 오조작으로 인해 소중한 트래픽 데이터(링크)가 영구 삭제되는 것을 방지**하기 위해, **삭제 직후 화면 하단에 5초간 떠오르는 토스트(Toast) 프롬프트에서 [실행 취소] 버튼을 클릭해 해당 링크를 완전히 복구**한다.

### [링크 클릭 성과 확인 (Analytics)]
- 사용자는 **자신이 개설해 둔 개별 링크들에 외부 방문자들이 각각 얼마나 반응했고, 클릭 점유율이 어떻게 되는지 성과를 직관적으로 파악**하기 위해, **관리 대시보드 화면상에서 각 활성 링크 블록의 우측에 노출되어 있는 '누적 클릭 수치(카운터)' 숫자를 읽고 확인**한다.
