# 마이링크 (MyLink) 기능 정의서 (PRD)

## 1. 프로젝트 개요
- **프로젝트명**: 마이링크 (MyLink)
- **목적**: 개발자, 크리에이터, 보안 전문가 등의 타겟 유저가 운영하는 채널 및 포트폴리오 링크를 하나의 프로필 페이지로 모아 직관적으로 공유할 수 있는 심플한 멀티 링크 서비스.
- **대상 사용자**: 복잡한 꾸미기 기능을 배제하고, `shadcn/ui` 기반의 깔끔하고 전문적인 인터페이스를 선호하는 집단.

## 2. 세부 요구사항 (MVP Core)

### 2.1. 사용자 인증 (Auth)
- **Firebase Google Login 단일화**: 이메일 등 별도의 계정 생성 과정 없이, Firebase Authentication을 활용한 구글(Google) 계정 소셜 기반 로그인만 필수 지원.

### 2.2. 프로필 관리 및 데이터베이스 구조
- **프로필 정보 구성**: 
  - **표시 이름 (displayName)**: 서비스의 URL 슬러그(Slug)이자 화면 전면에 표시되는 고유 타이틀. 소셜 로그인 시 **유저의 구글 지메일(Gmail) 아이디 앞부분을 파싱하여 기본값으로 자동 설정**. 
  - **사용자 이름 (username)**: 실제 사용자의 실명이나 본명을 나타내는 보조 정보 필드.
  - **소개글 (Bio)**: 프로필 전면에 노출하는 짧은 자기소개 텍스트.
- **단일 URL 라우팅**: 식별자인 `displayName` 값 자체를 퍼블릭 접근 경로(Path)로 즉시 활용. (예: `domain.com/displayName`)
- **프로필 정보수정 (인라인 편집)**: `displayName`, `username`, `Bio` 텍스트를 변경할 때 다른 페이지로 이동하지 않고 텍스트를 마우스로 직접 클릭하여 폼으로 변환시키는 **인라인 편집(Inline Editing)** 적용.
- **데이터베이스 모델링 (Firebase Firestore)**:
  - 계정 도큐먼트(Document) 구조 내부에 `displayName`과 `username` 필드를 필수로 보관.
  - 각 링크 정보들은 해당 유저 식별 도큐먼트 하위의 **서브 컬렉션(Subcollection)** 계층으로 안전하게 트랜잭션 관리.

### 2.3. 링크 관리 기능 (CRUD) 
- **링크 등록 및 삭제 체계**: 외부로 점프할 `URL` 문자열과 화면상 버튼 안에 표기될 `표시 제목(Title)` 기입 및 삭제 처리.
- **삭제 실행 취소(Undo) 안전장치**: 링크 삭제 아이콘 클릭 시 즉시 리스트에서 지워지나, 하단에 5초간 떠오르는 '토스트 알림(Toast)'의 [실행 취소] 액션을 통해 오조작 복구 가능.
- **인라인 편집 지원**: 각각의 링크 제목 및 연결 URL 텍스트 역시 그 자리에서 즉각 수정되는 전면적 인라인 편집 허용.
- **구글 파비콘 API (Favicon Parsing)**: 새 URL 삽입 시 **구글 파비콘 API**가 내부 호출. 원격지 로고 썸네일을 반환받아 서비스 내 해당 링크의 좌측 아이콘(Icon)으로 영구 적용.

### 2.4. 링크 행동 트래킹 및 통계 (Analytics)
- **개별 링크 클릭 집계 (Click Count)**: 사용자가 자신의 화면에 등록해 둔 링크 버튼이 외부 방문자에 의해 외부로 링크 아웃(클릭) 될 때마다 트리거를 걸어 해당 링크의 1회 조회 클릭 스코어를 누적 카운팅.
- **클릭 성과 대시보드 노출**: 소유자의 링크 관리 대시보드 화면상에서, 각 활성 링크 리스트의 우측 단에 해당 전용 링크가 총 몇 번의 클릭 전환을 일으켰는지 명확한 카운트(숫자)를 직관적으로 실시간 노출함.

### 2.5. 퍼블릭 공유 페이지 (Public View)
- **접속 뷰**: `domain.com/displayName` 포맷으로 렌더링.
- **디자인 시스템**: 디자인 테마 변경 기능을 배제. 코어 설계된 `shadcn/ui` 기반 단일 디자인 컴포넌트 생태계로 시각적 퀄리티를 통합/유지.

## 3. 기능 제외 대상 프로세스

### 3.1. MVP 명시적 제외 스펙 (Not In Scope)
- 프로필 뷰어/방문자에 대한 전역 단위 광범위 통합 방문자 통계 화면
- 프로필 이미지 파일 업로드 솔루션 (파비콘과 폰트 전용 텍스트만으로 디자인 완벽 대체)
- 구동 시 별도 인증 폼을 통해 가입되는 복구 불가능한 영구 아이디(ID) 메커니즘 
- 유저 자율도를 극대화하는 광범위 테마, 모서리, 배경 색상 디자인 커스터마이징 편집 도구
- 관리자 패널 창 내부에서 자체적으로 구현하는 모바일 뷰 미리보기(Preview) 렌더링 기능
- 등록된 개별 링크에 대해 강제적인 숨김/표시를 지원하는 (On/Off 토글 버튼) 형태 
- 마우스 드래그 앤 드롭을 구현하기 위한 복잡도 높은 순서 변경 제어 로직 구현
- 개발, 보안 등 특정 기술 분과 직군용만으로 특별히 편향 제작된 전용 부가 위젯 컴포넌트
